## Online Media Converter v(0.2.x):

## Pastanetwork Wiki Manager (v0.2.x):

<img src="https://www.pastanetwork.com/static/assets/img/logo.svg" width="200">

### Wiki Manager created for [Pastanetwork](https://www.pastanetwork.com) on NodeJS using Express and html/js/css files. 

Express is this case is serving both :
- API
- HTML/JS/CSS files 
- public assets (favicon.ico, sitemap.xml, robots.txt)

Service run on a Docker container and needs a PostgreSQL database to work. <br>It can be updated with included bash script.


**[Repository](https://github.com/pastanetwork/pasta-wiki-back)**

## Tirnatek Radio (v3.0.x) :

<img src="https://www.tirnatek.fr/DefaultIMG.png" width="200">

### Complete webradio interface, proxy & API for Azuracast
- **NextJS** as fronted ;
- **Express** as backend & proxy for **Azuracast** server ;
- **DiscordJS** with integrated server and private message bot;
- Upgradable & configurable via **Bash Script**;
- Deployed on a **Docker Compose network**.

**[Repository](https://github.com/St4lV/tntr-next-js)** - **[Website](https://www.tirnatek.fr/)**
